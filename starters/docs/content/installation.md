Add `@nuxtjs/content` dependency to your project:

```bash
yarn add @nuxtjs/content # npm install @nuxtjs/content
```

Then, add `@nuxtjs/content` to the `modules` section of `nuxt.config.js`:

```js
{
  modules: [
    '@nuxtjs/content'
  ],
  content: {
    // Options
  }
}
```
